# Creative.io - Modern Fluent Portfolio

A sleek, responsive, and high-performance single-page portfolio template built using semantic HTML, Tailwind CSS, and vanilla JavaScript. It features an ultra-modern "Fluent Glassmorphism" aesthetic with dynamic light/dark mode toggling, smooth ambient glow accents, and elegant entrance animations.

## ✨ Features

* **Fluent Glassmorphism UI:** Backdrop filters, subtle borders, and shifting ambient glow elements that create depth.
* **Dynamic Theme Toggle:** Full light and dark mode integration mapping custom CSS variables seamlessly across elements, with system memory via `localStorage`.
* **Fully Responsive Design:** Tailored layouts for mobile, tablet, and desktop viewports, complete with an interactive mobile slide-down menu.
* **Fluid Motion & Animations:** Leverages `Animate.css` for structural component entrances and custom hardware-accelerated Tailwind transitions for interactive micro-interactions.
* **Interactive Contact Form:** Clean, pre-styled modal-style section setup for instant client reach-outs.

## 🛠️ Built With

* **Structure:** HTML5 (Semantic Elements)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) (CDN Version)
* **Icons:** [Font Awesome v6.4.0](https://fontawesome.com/)
* **Animations:** [Animate.css](https://animate.style/)
* **Interactivity:** Vanilla JavaScript (ES6+)
